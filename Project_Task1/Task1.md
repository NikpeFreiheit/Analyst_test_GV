# User Story 
| Роль/Тип пользователя | Действие и ожидаемый результат |
|--|--|
| Руководитель магазина | Хочу получать отчет по оборотам в магазине, состоянию товаров/заказов |
| Главный бухгалтер | Хочу получать отчет по оборотам в магазине |
| Заведующий складом | Хочу получать отчет по состоянию товаров/заказов |
| Оператор/Администратор онлайн-магазина | Хочу создавать, изменять заказ онлайн.|
| Клиент/Покупатель | Хочу получать заказ на свой адрес; Оставлять комментарий к заказу, товару; Создавать, изменять, отменять заказ онлайн; Управлять содержимым ЛК; Оплачивать товар онлайн в онлайн-магазине; Получать кредит/рассрочку на товар онлайн без отдельного обращения в банк |
| Оператор склада | Вносить, изменять, удалять информацию о товаре |
| Бухгалтер | Вносить, изменять, удалять бухгалтерскую информацию  |
| Сотрудник по работе с товарами в ИС | Создавать, изменять, отменять заказ; Управлять содержимым ЛК клиента; Отвечать на комментарии к товарам и заказам; Отвечать о решении проблемы; Создавать, отправлять, получать email и смс; Передавать на склад запросы о необходимых товарах (закупке, поставке); Передавать информацию о полученных товрах и их качестве. |
| Сотрудник доставки | Передавать информацию о факте доставки/отказа; Получать данные для доставки. |
| Поставщик банковских услуг | Получать и передавать информацию о клиенте для оценки возможности выдачи кредита/рассрочки; Передавать информацию о совершённых транзакциях оплаты товара/возврата средств при отказе от товара, выпущенных картах, открытых счетах и балансе на счете; Осуществлять транзакцию через подключение к онлайн магазину без перехода в систему банка |
|Внешние системы-источники новых клиентов| Обмениваться информацией о портрете клиента/потенциального клиента, его потребностях и интересах на основе состава заказов и избранных товаров; Получать и передавать контактные данные клиента для осуществления рекламы|



# Словарь предметной области
|Термин|Описание|
|--|--|
| Онлайн-магазин | Система с GUI в базой данных для хранения и управления информацией о клиентах |
| ИС по работе с товарами | ИС управляющая заказами, поставками товаров, ценами товаров, отправкой товаров в достаку |
| Службы доставки | Внешние системы со своими приложениями, интеграция с которыми позволяет пользоваться услугами доставки |
| Поставщик банковских услуг | Внешняя система со своим приложением, интеграция с которым позволяет получать банковские услуги по оплате товаров, в том числе через кредитование/рассрочку |
| Внешние системы-источники новых клиентов | Внешние системы со своими приложениями, интеграция с которыми позволяет обмениваться информацией для рекламы и привлечения новых клиентов |
| Клиент | Физическое лицо, посетившее онлайн магазин и зарегистрировавшееся в ЛК |
